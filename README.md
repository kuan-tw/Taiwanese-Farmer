# Taiwanese Farmer 逮丸農夫

一個可以查詢農作物時價的Discord 機器人  

# 用法
### **!hello**  |   跟阿伯打招呼
### **/hello**  |   跟阿伯打招呼
### **/help**      |     顯示指令列表
### **/ping**    |   顯示機器人的延遲
### **/invite**  |   獲取機器人邀請連結
### **/crops [農作物]**  |   查看指定農作物的資訊
### **/price [農作物]**  |   查看指定農作物的當日價格
